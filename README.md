
# ToDo List App

This is a simple ToDo list app created using React. You can add tasks, mark them as completed, and delete them.

## Live Demo

Check out the live demo of the ToDo list app hosted on Netlify: (https://jocular-empanada-f434f6.netlify.app)

## Installation

To run this app locally, follow these steps:

1. Clone this repository to your local machine:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd <project-directory>
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the app.

## Usage

- Enter a task in the input field and press "Enter" or click the "Add" button to add it to the list.
- Click on the checkbox next to a task to mark it as completed.

## Files

- `App.js`: Contains the main component of the ToDo list app.
- `ToDoItem.js`: Contains the component for rendering individual ToDo items.
